# Card sorting

## When to Use

Activate this skill when:
- Working on card sorting tasks
- Reviewing or improving existing card sorting implementations
- Troubleshooting issues related to card sorting
- Setting up or configuring card sorting from scratch

## Workflow

1. Define research questions or product hypotheses
2. Select appropriate method (interviews, surveys, analytics, etc.)
3. Recruit or identify participants or data sources
4. Execute research or analysis plan
5. Synthesize findings into actionable insights
6. Prioritize insights by impact and confidence
7. Communicate findings to stakeholders
8. Define success metrics and measurement plan
9. Close the loop: validate decisions against outcomes

## Quality Bar

- Research questions answered with sufficient evidence
- Insights tied to business metrics
- Decisions documented with supporting evidence
- Success metrics defined before launch
