# React native

## When to Use

Activate this skill when:
- Working on react native tasks
- Reviewing or improving existing react native implementations
- Troubleshooting issues related to react native
- Setting up or configuring react native from scratch

## Workflow

1. Review platform-specific requirements (iOS/Android guidelines)
2. Design UI following platform conventions (HIG/Material Design)
3. Implement features with offline-first patterns where needed
4. Handle app lifecycle events and background tasks
5. Optimize for performance, battery life, and memory usage
6. Test on physical devices across supported OS versions
7. Review for accessibility (VoiceOver/TalkBack)
8. Address crash reporting and error monitoring
9. Prepare for app store submission (metadata, screenshots, review guidelines)

## Quality Bar

- Passes all platform-specific tests
- Meets platform accessibility standards
- Offline scenarios handled gracefully
- App store guidelines compliant
- Crash-free rate meets target (e.g. >99.9%)
